Projects Introduction

      In the past two years, I've done three innovative projects that showcase my expertise in software development, compiler design, and game development. The Random Lifestyle Generator offers users a customizable solution to inject spontaneity into their daily routines through a user-friendly Java GUI interface. The Compiler Individual Project demonstrates my ability to translate human-readable language into machine code efficiently using ALGOL C language, incorporating sophisticated symbol table functionality for informed decision-making. Finally, the Objective-Oriented Programming Individual Project introduces the Battle Bots Game, an immersive virtual arena where autonomous characters engage in intense combat, highlighting my proficiency in artificial intelligence and user interface design.

Project 1 -- Lifestyle Generator

      This is a versatile software development project implemented in Java with a GUI interface, designed to add an element of spontaneity to users' daily routines. With customizable wheels and the ability to effortlessly add, edit, and delete options, users can tailor their experiences to suit their preferences. Project 2 -- Compiler

      This is a sophisticated system designed to translate human-readable language into machine code using ALGOL C language, leveraging Lex and YACC generators. By parsing user inputs and adhering to predefined rules within lex and YACC files, the compiler efficiently generates machine code. Furthermore, the incorporation of a symbol table enables meticulous tracking of existing expressions and variables, facilitating informed decisions regarding the availability of adding, naming, and deleting functions

Project 3 -- Battle Bots Game

      This is an immersive virtual arena where small yet mighty characters engage in intense combat autonomously. As the battle commences, these ball-like avatars navigate the arena, collecting coins and launching attacks without direct player control. With a user-friendly interface enabling players to select between one to five characters for each match, the game offers dynamic and customizable gameplay experiences.
